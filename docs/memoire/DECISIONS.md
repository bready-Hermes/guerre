# Décisions projet — guerre

- Créer un site statique autonome, sans backend ni base de données.
- Ton : documenté, accessible, sérieux, non complotiste, non sensationnaliste.
- Message central : le risque existe, il augmente avec la polarisation et l’érosion des traités, mais il dépend aussi de garde-fous politiques, militaires, diplomatiques et citoyens.
- Publier selon le workflow standard Yves : dev sur `guerre-dev.notreserveur.net` branche `hermes`, prod sur `guerre.notreserveur.net` branche `main`.
